# FINANCIAL-DOCUMENT-MANAGEMENT-SYSTEM
This is a FastAPI project for managing financial documents.

Users can:
- Register
- Login
- Upload PDF files
- Search documents
- Delete documents
- Perform semantic search using RAG

#Technologies Used
- Python
- FastAPI
- FAISS
- LangChain
- JWT Authentication

Run Project
Install dependencies:
pip install -r requirements.txt
Run server:
uvicorn main:app --reload
Open Swagger UI:
http://127.0.0.1:8000/docs
